SCSS 命名規範 (BEM設計模式) :<br>
<blockquote>
  區塊 (block) .block<br>
  元素 (element) .block__element<br>
  修飾 (modifier) .block__element--modifier<br>
</blockquote>
<br>
JS變數命名規範 :<br>
<blockquote>
  駝峰式命名<br>
  全大寫：全域變數<br>
  變數前$：jQuery變數<br>
  變數前_：外部變數<br>
</blockquote>
